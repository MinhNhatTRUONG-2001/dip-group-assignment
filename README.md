# Data-Intensive Programming Group Assignment
These are my works in the group assignment of Data-Intensive Programming course at Tampere University. Apache Spark, Scala and Databricks were used. Note that I only include my contribution to the assignment and do not include other teammates' works.

The datasets can be found [here](https://drive.google.com/drive/folders/1XApx-5MLYqFByelYjBV1bZbHHx8oqVc0?usp=sharing).

## What were done in the assignment?
- The basic tasks were some data analytics of video game sales and football datasets.
- The advanced task 2 was text processing from some Wikipedia articles.
- The advanced task 4 was using Spark's machine learning packages to make a predict from a large data stream of energy dataset 'kampusareena'.
